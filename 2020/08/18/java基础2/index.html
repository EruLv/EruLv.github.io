8.接口

1.抽象方法和抽象类

仅有声明而没有方法体。


abstract class Instrument {    //抽象类
  private int i; // Storage allocated for each
  public abstract void play(Note n);    //抽象方法
  public String what() { return "Instrument"; }
  public abstract void adjust();
}

包含抽象方法的类叫做抽象类。如果一个类包含一个或多个抽象方法，则该类必须是抽象的。

如果从一个抽象类继承，必须为基类中的所有抽象方法提供定义，否则该类也会被强制认为是抽象类。

2.接口

更纯粹的抽象方法，接口内不能有方法体。

用interface关键字代替class

要想某个类遵循特定的接口，需要用implements关键字（类似于extents，必须实现接口的全部方法。

接口类中的方法必须被定义为public（默认public）。

interface Instrument {
  int VALUE = 5; 
  void play(Note n); // Automatically public
  void adjust();
}

class Wind implements Instrument {
  public void play(Note n) {
    print(this + ".play() " + n);
  }
  public String toString() { return "Wind"; }
  public void adjust() { print(this + ".adjust()"); }
}

3.多重继承

在implements可以添加多个接口，用逗号隔开。

当同时有继承操作时，需要把extends放在前面。

interface CanFight {
  void fight();
}

interface CanSwim {
  void swim();
}

interface CanFly {
  void fly();
}

class ActionCharacter {
  public void fight() {}
}  

//继承+接口
class Hero extends ActionCharacter
    implements CanFight, CanSwim, CanFly {
  public void swim() {}
  public void fly() {}
}

4.通过继承拓展接口

接口与类一样也可以继承。

interface Monster {
  void menace();
}

interface DangerousMonster extends Monster {
  void destroy();
}

interface Lethal {
  void kill();
}

class DragonZilla implements DangerousMonster {
  public void menace() {}
  public void destroy() {}
}  

interface Vampire extends DangerousMonster, Lethal {
  void drinkBlood();
}

class VeryBadVampire implements Vampire {
  public void menace() {}
  public void destroy() {}
  public void kill() {}
  public void drinkBlood() {}
}

5.接口中的域

接口中的域默认时static和final的，即全局常量。

public interface RandVals {
  Random RAND = new Random(47);
  int RANDOM_INT = RAND.nextInt(10);
  long RANDOM_LONG = RAND.nextLong() * 10;
  float RANDOM_FLOAT = RAND.nextLong() * 10;
  double RANDOM_DOUBLE = RAND.nextDouble() * 10;
} 

public class TestRandVals {
  public static void main(String[] args) {
    print(RandVals.RANDOM_INT);
    print(RandVals.RANDOM_LONG);
    print(RandVals.RANDOM_FLOAT);
    print(RandVals.RANDOM_DOUBLE);
  }
} /* Output:

6.嵌套接口

在类中声明接口，接口是static的，其他类可以通过类名访问接口。类内的接口可以设置为默认,public，private。

class A {
  interface B {
    void f();
  }
  public class BImp implements B {
    public void f() {}
  }
  private class BImp2 implements B {
    public void f() {}
  }
  public interface C {
    void f();
  }
  class CImp implements C {
    public void f() {}
  }    
  private class CImp2 implements C {
    public void f() {}
  }
  private interface D {
    void f();
  }
  private class DImp implements D {
    public void f() {}
  }
  public class DImp2 implements D {
    public void f() {}
  }
  public D getD() { return new DImp2(); }
  private D dRef;
  public void receiveD(D d) {
    dRef = d;
    dRef.f();
  }
}
