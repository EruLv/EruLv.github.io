距离上次使用java已经过去2年多了，趁大三夏令营结束的这段时间复习一下java.

1.一切皆是对象

  1. ava是一种更纯粹的面向对象的编程语言。

  2. Java使用new关键字创建对象，对象的存储位置在堆内。对于一些基本类型，new对堆的操作开销较大，因此对于这些类型，java采取与c,c++相同的方法，即，不用new来创建这些变量，而直接创建一个并非是引用的变量，这个变量直接存储值，并置于堆栈中，音因此更加高效。

  3. Java变量的基本类型:

9.static

当一个事物声明是static时，意味着这个域或方法不会与包含它的那个对象关联在一起。即无论创建多少个包含它的对象，系统只给它分配一个存储空间。

class test{
    static int a = 1;
}

class Main {
    public static void main(String args[]){
        test t = new test();
        t.a ++;
        test tt = new test();
        tt.a ++;
        test.a ++;    //可以直接通过类名调用静态方法
        System.out.println(t.a);
        System.out.println(tt.a);
    }
}

2.操作符

1.赋值

基本类型的赋值:

直接把内容赋值给左边，如a = b, 当改变b的值时，a的内容不变。

对象赋值:

把指向对象的引用复制给左边，如c = d,那么c和d都指向同一存储位置。

2.random

import java.util.*;

class Main {
    public static void main(String args[]){
        Random random = new Random(66);    //随机数对象，种子:66
        int a = random.nextInt(100) + 1;    //设置Int范围1-100
        float b = random.nextFloat()*10;    //设置float范围0-10
        System.out.println(a);
        System.out.println(b);
    }
}

4.标签

break label: 跳出标签所指循环。

continue label:跳过标签所指循环的这一步。

class Main {
    public static void main(String args[]){
        int i = 0;
        outer:
        for(;;){
            inner:
            for(;i < 10; i++){
                print("i = "+i);
                if(i == 5) break outer;    //直接跳出外部循环
            }
        }
    }
}

3.this

在类内使用，表示该类的引用。可用于区分形参和类内参数。

一般的方法，如:

class Person {
    public void eat(Apple apple) {
        print("Yummy");
    }
}

//实际上隐含的代码是(实际不能把this写上，否则编译器报错):
class Person {
    public void eat(this, Apple apple) {
        print("Yummy");
    }
}

4.再看static

static方法是没有this的方法，在所以在static方法内部不能直接调用非静态方法，单反过来是可以的。

static很像全局方法，给java提供了一种访问全局变量和方法的思路。
